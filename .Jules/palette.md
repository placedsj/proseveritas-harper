## 2025-05-20 - Interactive Cards Accessibility
**Learning:** Clickable cards implemented as `div`s are not keyboard accessible.
**Action:** Use `<button>` elements for interactive cards, ensuring `w-full text-left` classes are applied to maintain layout.

## 2025-05-21 - Global Search Shortcuts
**Learning:** Adding keyboard shortcuts (Cmd+K) and standard modal behaviors (Esc to close) significantly improves power user workflow and accessibility.
**Action:** Always include keyboard shortcuts for global actions and ensure modals handle Escape and backdrop clicks.
